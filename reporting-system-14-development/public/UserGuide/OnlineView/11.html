ANALYSIS
The following section explains the
Analysis
option in the left menu:
1.
Office
– select a specific department
2.
Year
– type in a specific year
3.
Branch
– select the branch
4.
Status
– select the status of the reports to be analyzed
5.
Display Office Names
– shows for each month which department
submitted the report and its status
6.
Show Sa’iqin Report
– shows the status of ‘Amila meetings, Ijlas ‘Am
meeting and Sa’iqin activity report
