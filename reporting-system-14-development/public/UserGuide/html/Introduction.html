Introduction

Purpose

The Online Reporting System is built to facilitate the timely submission of departmental reports and other administrative functions. This User Guide provides the reader with the information needed to use this system.

 

What is needed?

Each office bearer needs to have a User ID, and a Password.  This information is provided by the General Secretary of the Jama‘at or the National General Secretary and should be kept confidential.
